## What is it?

After each analysis, Morphiq Labs surfaces specific issues found across your crawled pages. Each issue is tied to a scoring dimension and assigned a priority so you know what to fix first.

---

## Issue categories

### Technical Structure

- **Schema Markup** (impacts up to 40 pts) — Missing or incomplete JSON-LD, irrelevant types, low coverage
- **Metadata** (impacts up to 30 pts) — Missing titles, descriptions, canonical URLs, Open Graph, or Twitter Cards
- **FAQ Sections** (impacts up to 20 pts) — No FAQ content on eligible pages
- **Content Quality** (impacts up to 10 pts) — Thin pages (under 300 words or fewer than 3 paragraphs)

### AI Visibility

- **llms.txt** — Missing or outdated AI-facing site index
- **Citation Signals** — Weak citation presence across AI model sources
- **AI Content Optimization** — Pages not structured for AI model citation

### Brand Positioning

- **Authority Building** — Opportunities to strengthen brand authority signals
- **Brand Messaging** — Sentiment optimization for how AI models describe you

---

## Discovery tiers

Issues are grouped by severity:

- **Fundamental** — Critical gaps that significantly hurt your scores
- **Intermediate** — Important improvements with measurable impact
- **Advanced** — Optimization opportunities for competitive edge
- **Polish** — Fine-tuning for marginal gains

---

## Fixing issues with one click

Every issue with an assigned agent shows a **Fix** button. Click it and Morphiq Labs:

1. Deploys the right agent for that issue type
2. Generates the fix (schema markup, meta tags, FAQ content, etc.)
3. Creates a GitHub pull request with the changes — or generates code you can copy if GitHub is not connected
4. Updates the issue status as the agent works

You review and merge the PR. That's it.

---

## Issue statuses

- **Identified** — Issue found, ready to fix
- **In Progress** — Agent is working on it
- **Completed** — Fix generated successfully
- **Merged** — PR merged, issue resolved

---

## No-code platforms

If you are on Webflow, Shopify, or another no-code platform, Morphiq Labs generates the code for you to paste instead of creating a PR. Click **Generate Code** to get copy-ready output.
